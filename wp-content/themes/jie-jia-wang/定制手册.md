## 四、注意事项

### 1. 告知客户事项

#### 关于域名和服务器
- 域名需要客户自行购买或委托购买，费用另计
- 服务器/虚拟主机需要客户自行购买或委托购买
- 推荐配置：PHP 7.4+、MySQL 5.7+、支持HTTPS

#### 关于内容维护
- 网站交付后，客户可通过WordPress后台自行更新内容
- 服务项目、客户评价可在后台直接添加/编辑
- 企业信息在"外观 → 自定义"中修改
- 提供操作指导文档或视频

#### 关于版权
- 客户提供的图片、文字需确保拥有版权
- 如需使用网络图片，建议使用免费商用图库
- 网站代码基于MIT协议，可自由使用

#### 关于售后
- 交付后7天内免费修复bug
- 后续维护需签订维护协议
- 功能新增按增值服务收费

### 2. 开发注意事项

#### 技术限制
- 主题基于WordPress开发，需WordPress 5.0+环境
- 不支持多语言（如需需额外开发）
- 不支持在线支付（如需需额外开发）
- 不支持会员系统（如需需额外开发）

#### 配色修改
- 修改配色需编辑CSS变量，建议由技术人员操作
- 配色方案需考虑可访问性和品牌一致性
- 建议提供品牌色HEX值

#### 内容更新
- 服务项目、客户评价通过后台管理
- 企业信息通过WordPress自定义器修改
- 图片通过媒体库上传

### 3. 常见问题处理

| 问题 | 解决方案 |
|-----|---------|
| Logo上传后仍有方块 | 检查是否正确上传，清除缓存 |
| 导航链接跳转404 | 确保页面存在，检查固定链接设置 |
| 图片显示模糊 | 上传更高分辨率图片 |
| 手机端显示异常 | 清除浏览器缓存，检查CSS加载 |
| 后台无法编辑 | 检查用户权限，确保是管理员角色 |

### 5. AI定制指南

#### 修改配色方案

编辑 `style.css` 中的 `:root` 变量：

```css
:root {
    --terracotta-500: #新颜色值;  /* 主题强调色 */
    --brown-800: #新颜色值;       /* 主要文字色 */
}
```

#### 添加新的自定义设置

在 `functions.php` 的 `jiejiawang_theme_customizer()` 函数中添加：

```php
$wp_customize->add_setting('new_setting', array(
    'default' => '默认值',
    'sanitize_callback' => 'sanitize_text_field',
));
$wp_customize->add_control('new_setting', array(
    'label' => '设置名称',
    'section' => 'jieja_config',
    'type' => 'text',
));
```

在模板文件中调用：

```php
$new_value = get_theme_mod('new_setting', '默认值');
```

在 `functions.php` 中扩展 `jiejiawang_service_meta_box_callback()` 函数：

// 添加新字段
$new_field = get_post_meta($post->ID, '_new_field', true);

function jiejiawang_save_service_meta($post_id) {
